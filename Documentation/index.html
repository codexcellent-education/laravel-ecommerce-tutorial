Folder Structure


1. Once You Install The Product on Your Server, You Will have Two Folder and Few Files.

2. Folders are "assets" and "core", In assets folder We keep assets like CSS, JS, Images. You can Edit Or Change Any Asset.

3. On Folder "core", We keep The Core of Laravel.

4. We hardly maintain Laravel Structure(MVC) on This Project.

5. "core/routes/web.php" All Routes (URL) in This Folder. You can find the ControllerName@FunctionName Here.

6. "core/app" All Models are in This Folder.

7. "core/app/Http" All Controller are in This Folder. You can Change anything you want if you need to change any functionality.

8. "core/resources/views" You Have All Blade file Here. You can Change anything from changing blade file.

Setup Payment Gateway


In Payment Gateway Panel You need to update your Payment Gateway Settings.

Here Have 6 Payment Methods.

1. Paypal

2. Perfect Money

3. Bitcoin - (Blockchain)

4. Strie - (Card)

5. Coinpayment - (Crypto Currency Coain)

6. COD - (Cash On Delivery)



Setup Paypal :

1. Click Paypal row Edit Gateway and Open a Modal.

2. Update with your Display Name, Display Image, Conversion Rate and Paypal Email.

3. Click Update and Update It.



Setup Perfect Money :

1. Click Perfect Money row Edit Gateway and Open a Modal.

2. Update with your Display Name, Display Image, Conversion Rate, Perfect Money USD ID and Alternative Passphrase.

3. Click Update and Update It.



Setup Bitcoin - (BlockChain) :

1. Click Bitcoin - (BlockChain) row Edit Gateway and Open a Modal.

2. Update with your Display Name, Display Image, Conversion Rate, Blockchain API kry and XPUB Key..

3. Click Update and Update It.



Setup Stripe - (Card) :

1. Click Stripe - (Card) row Edit Gateway and Open a Modal.

2. Update with your Display Name, Display Image, Conversion Rate, Secret Key and Publisher Key.

3. Click Update and Update It.



Setup Skrill - Moneybooker :

1. Click Skrill row Edit Gateway and Open a Modal.

2. Update with your Display Name, Display Image, Conversion Rate, Merchant Email and Secret Key.

3. Click Update and Update It.



Setup Coinpayment :

1. Click Coinpayment row Edit Gateway and Open a Modal.

2. Update with your Display Name, Display Image, Conversion Rate, Merchant Key, IPN Secret.

3. Click Update and Update It.



Setup COD - (Cash On Delivery) :

1. Click COD - (Cash On Delivery) row Edit Gateway and Open a Modal.

2. Update with your Display Name, Display Image, Conversion Rate

3. Click Update and Update It.



Setup Mail Driver


Open .env File and edit bellow instruction.


Send email with PHP Mailer :



You just replace MAIL_FROM_ADDRESS & MAIL_FROM_NAME to your sending email and name.

Send email with SMTP Server :



You need to replace this
MAIL_HOST=smtp-host
MAIL_PORT=smtp-port
MAIL_USERNAME=smtp-username
 MAIL_PASSWORD=smtp-password
MAIL_ENCRYPTION=smtp-encryption
MAIL_FROM_ADDRESS=your-sending-email
MAIL_FROM_NAME=your-sending-name
